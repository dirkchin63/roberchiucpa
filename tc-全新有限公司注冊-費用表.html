成立全新香港有限公司的程序

  1. 客戶自行設定公司名稱
  2. 客戶填寫 網上申請表格 (請按此下載) 後，
    連同該公司股東 / 董事 / 公司秘書之身份證 或 護照副本 和 地址証明交給我們
    (請電郵到 info@profitaccounting.hk 或 傳真到 (852) 2854 1788)
  3. 我們將查冊客戶採用的公司名稱是否可註冊
  4. 我們將派專人與客戶聯絡，驗證客戶身份
  5. 所需資料完成審核後，我們會電郵賬單給客戶
  6. 客戶付款後，我們將於2小時內準備妥正式註冊文件
  7. 客戶可親身到本公司簽署正式註冊文件，或我們可電郵有關文件給客戶，客戶簽署後交回正本我們
  8. 我們向公司註冊處遞交正式註冊文件
  9. 公司註冊證書、商業登記証和公司綠盒 完成
    (須時約7個工作天完成, 受限於公司註冊處所需的處理時間)
  10. 我們將通知客戶領取有關公司文件及公司文件盒
    客戶可到本公司領取 或 我們將有關文件及公司文件盒速遞給客戶。

註解 1 - 公司秘書服務 (已包括政府費用 HK$105)

  1. 由擁有專業資格及經驗豐富的「香港特許秘書公會」資深會員 和「特許公司治理公會」的 特許秘書 主理，令客戶不用操心 貴公司之合規事宜。
  2. 盈大會計有限公司代表出任法定公司秘書職位
  3. 為 貴公司準備及提交 周年申報表 (NAR1表格) 給公司註冊處
  4. 維護及更新 貴公司的「成員登記冊」、「董事登記冊」及「公司秘書登記冊」
  5. 向客戶就公司秘書事宜提供咨詢和意見。
  6. 其他額外公司秘書服務及政府費用，本公司會另行收取

註解 2 -「指定代表」服務 和「重要控制人登記冊」存檔服務

自 2018 年 3 月 1 日 起，在香港註冊成立的公司必須指定一名人士為該公司的「指定代表」，以提供該公司的「重要控制人登記冊」的有關協助予執法人員。在香港註冊成立的公司必須在公司的註冊辦事處或在香港的其它地方備存「重要控制人登記冊」和 允許執法人員檢查「重要控制人登記冊」。

凡未有履行備存「重要控制人登記冊」責任，即屬刑事罪行。有關「指定代表」和 備存「重要控制人登記冊」的本公司簡短擇錄，請點擊「香港有限公司 - 註冊條件」。有關備存「重要控制人登記冊」的詳情，請參閱「香港公司註冊處」的「對外通告」及「公報」。

本公司的「指定代表」服務 和「重要控制人登記冊」存檔服務包括 :

  1. 盈大會計有限公司代表出任 貴公司的「指定代表」職位
  2. 把 貴公司的「重要控制人登記冊」備存在本公司的辦公室
  3. 向執法人員就 貴公司的「重要控制人登記冊」查詢提供協助

註解 3 - 公司註冊地址服務

  1. 使用中環 辦公室地址 (香港中環 德輔道中148號 安泰大廈 9 樓 全層) 作政府註冊 及 商業登記
  2. 中環被視為香港的中心商業區，租金一向高昂。我們的中環辦公室地址，有助你的公司樹立良好企業形象，開展生意。
  3. 使用中環 辦公室地址作 政府信件、商業信件 及 公眾通訊用途
  4. 代收 信件和包裹 (長+闊+高，不超過90cm)，並作出即時電郵通知客戶
  5. 客戶可選擇 :
    (i) 客戶可到本公司收取郵件, 或
    (i) 單次、每週定時 或 每月定時，轉寄郵件和包裹到客戶指定地址。
    (ii) 轉寄服務需另收費

註解 4 - 安排銀行開戶會議

  1. 現在香港，可能需要長達一個月的時間才能與銀行預約銀行開戶會議。
  2. 藉着我們的網絡以及與銀行的長期合作夥伴關係，我們可以協助客戶加快在以下銀行的其中一間，安排銀行開戶會議 。
    (匯豐銀行、恒生銀行、中國銀行、華僑銀行、上海商業銀行 或 Airwallex)
  3. 我們的公司成立服務，不包括銀行收取的開戶服務費。
  4. 銀行開戶是銀行的商業決定，本公司不保證銀行開戶的成功。
